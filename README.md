## 소개
안녕하세요. 나이가 들어도 게임 개발을 계속하고 싶은 개발자 이혁준입니다.<br/><br/>
저는 개발이라는 일에 국한되지 않고, 게임이라는 제품을 만드는 과정에 깊은 열정을 가지고 있습니다. 항상 새로운 것에 대한 두려움 없이, 마치 게임의 미션을 클리어하듯이 일에 임하고 있습니다.<br/><br/>
15년이라는 시간 동안 다양한 프로젝트를 진행하며 성장해왔습니다. 특히, 7개의 프로젝트를 처음 시작부터 서비스 런칭까지 경험하면서, 기획, 디자인, 개발, 테스트, 배포의 모든 과정을 이해하고 있습니다. 이 과정에서 팀원들과의 협업의 중요성을 깨닫고, 문제 해결 능력과 소통 능력을 더욱 강화할 수 있었습니다.<br/><br/>
저는 플레이어가 진정으로 즐길 수 있는 게임을 만들기 위해 항상 노력하고 있으며, 혁신적이고 창의적인 아이디어를 통해 새로운 경험을 제공하고자 합니다.
<br/><br/>
